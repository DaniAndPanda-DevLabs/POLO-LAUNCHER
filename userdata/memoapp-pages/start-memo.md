# 메모 앱에 오신 것을 환영합니다!

안녕하세요! 이 메모 앱을 사용해 주셔서 감사합니다.

이곳은 아이디어, 할 일, 공부 내용, 일기 등 무엇이든 자유롭게 기록할 수 있는 공간입니다.

## 주요 기능

* 빠르고 편안한 메모 작성
* 자동 저장
* 여러 페이지 생성 및 관리
* 다크 모드 / 라이트 모드
* 가볍고 쾌적한 사용 경험

## 사용 팁

* 메모는 자동으로 저장되므로 저장 버튼을 누를 필요가 없습니다.
* 페이지를 만들어 주제별로 메모를 정리해 보세요.
* Markdown 문법을 지원합니다. 자세한 사용법은 **Markdown 문법** 문서를 확인하세요.

생각이 떠오르는 순간, 바로 기록해 보세요.
