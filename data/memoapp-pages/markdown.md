# Markdown 문법 안내

이 문서는 메모 앱에서 사용할 수 있는 기본 Markdown 문법을 소개합니다.

---

# 제목

제목은 `#` 기호를 사용합니다.

```md
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6
```

---

## 강조

```md
**굵게**

*기울임*

~~취소선~~
```

결과

**굵게**

*기울임*

~~취소선~~

---

## 목록

### 순서 없는 목록

```md
- 항목 1
- 항목 2
  - 하위 항목
```

결과

* 항목 1
* 항목 2

  * 하위 항목

---

## 체크리스트

```md
- [ ] 해야 할 일
- [x] 완료한 일
```

결과

* [ ] 해야 할 일
* [x] 완료한 일

## 표

```md
| 이름 | 나이 | 직업 |
|------|-----:|------|
| 홍길동 | 20 | 학생 |
| 김철수 | 25 | 개발자 |
```

## 줄바꿈

문단을 구분하려면 한 줄을 비우세요.

```md
첫 번째 문단입니다.

두 번째 문단입니다.
```

---

## 지원하는 주요 문법

* 제목 (`#`)
* 굵게, 기울임, 취소선
* 순서 있는 목록
* 순서 없는 목록
* 체크리스트
* 링크
* 이미지
* 인라인 코드
* 코드 블록
* 인용문
* 구분선
* 표
* 문단 및 줄바꿈

Markdown을 활용하면 메모를 더욱 읽기 쉽고 체계적으로 정리할 수 있습니다.
